# 🍴 EatFit Clone

A front-end clone of the EatFit website built using **HTML & CSS**.  
This project replicates the design and layout of EatFit, focusing on clean structure, responsive styling, and practicing core web development fundamentals.

---

## 🚀 Features
- Static webpage with semantic HTML
- Styled using pure CSS (no frameworks)
- Responsive design for different screen sizes
- Organized sections for navigation, content, and footer
